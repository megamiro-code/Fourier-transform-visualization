# Fourier-transform-visualization
Fourier Visualizer (フーリエ変換可視化ツール)
このツールは、画像を二次元高速フーリエ変換（FFT）し、周波数領域での成分選択が「画像としてどのような波形（基底関数）に対応し、どのように合成されるか」を直感的に理解するためのインタラクティブ・デモです。

🚀 主な機能
リアルタイム FFT 解析: 画像を読み込むと即座に周波数マップ（パワースペクトル）を生成します。

インタラクティブな成分抽出:

周波数マップ上をブラシでなぞることで、特定の周波数成分のみを取り出せます。

「最後に選択した波形」: 直近でクリック・ドラッグした領域が、実空間でどのような「縞模様（波）」に対応するかを個別に表示します。

「合成画像」: 選択したすべての成分を重ね合わせ（逆フーリエ変換）、元の画像を再構築する過程を確認できます。

高解像度・高速処理: 256x256 の解像度で、Webブラウザ上で高速に動作します。

カスタマイズ可能なブラシ: 消しゴムモードやサイズ調整により、詳細な成分解析が可能です。

🛠 使い方
画像の読み込み:
「① 元画像」のエリアに画像をドラッグ＆ドロップするか、クリックしてファイルを選択します。

周波数成分の選択:
「② 周波数マップ」上をマウスでドラッグします。

中心部: 低周波成分（画像の全体的な明るさや、ぼんやりした形）。

周辺部: 高周波成分（エッジ、輪郭、細かいテクスチャ）。

結果の確認:

③ 最後に選択した波形: 今なぞった場所が「どんな向き・細かさの波」なのかを確認。

④ 合成画像: 塗りつぶした領域が増えるにつれ、元の画像が復元されていく様子を確認。

💡 技術的な特徴
アルゴリズム: Cooley-Tukey型 高速フーリエ変換 (FFT) 実装。

可視化:

周波数マップは対数スケールでスケーリングされ、微細なピークも視認可能。

選択領域を「青」、直近領域を「赤」でハイライト表示（内部的にはマスク処理）。

ゼロ依存関係: 外部ライブラリを一切使用せず、単一の HTML ファイル（HTML5 Canvas + Vanilla JS）で完結しています。

📂 ファイル構成
index.html: 本体コード（HTML/CSS/JS 全てを含む）
